### **Step 3: Use Zoho Catalyst for Heavy Logic (Not Deluge)**

**Deluge has limits.** For complex logic (ML, heavy calculations, optimization), use **Zoho Catalyst serverless functions**.

#### **Example: Asset Depreciation Function in Catalyst**

**Create a Catalyst function** (Node.js or Python):

```python
# Zoho Catalyst Function (Python)
import json

def asset_depreciation(cost, useful_life_years, category):
    """
    Calculate straight-line depreciation
    Input: asset cost, useful life, category
    Output: annual depreciation, residual value
    """
    annual_depreciation = cost / useful_life_years
    
    # Category-based adjustments
    if category == "IT":
        useful_life_years = 3  # IT depreciates faster
    elif category == "Furniture":
        useful_life_years = 5
    elif category == "Heavy Machinery":
        useful_life_years = 10
    
    annual_dep = cost / useful_life_years
    residual = cost * 0.1  # 10% residual value
    
    return {
        "annual_depreciation": annual_dep,
        "residual_value": residual,
        "months_remaining": useful_life_years * 12
    }
```

**How to use from Creator:**

1. **Write the function in Catalyst** (use your AI IDE to vibe-code it)
2. **Deploy to Catalyst** via Catalyst CLI
3. **Call from Creator's Deluge:**
   ```deluge
   response = zoho.catalyst.invokeFunction("asset_depreciation", {
       "cost": asset.get("Cost"),
       "useful_life_years": 5,
       "category": asset.get("Category")
   });
   depreciation_amount = response.get("annual_depreciation");
   ```

4. **Update Creator record** with the result
5. **Zoho Flow triggers** this entire workflow on asset creation

**Result: Complex logic lives in Catalyst, Creator orchestrates, Flow automates.**[6][7]
